# Setup

  • Just install @aspectjs/core, along with the required polyfills if needed:
    yarn add @aspectjs/core
## install polyfills if needed
yarn add reflect-metadata

    npm install --save @aspectjs/core
## install polyfills if needed
npm install --save reflect-metadata'

    // Make sure to add code blocks to your code group
    • import the polyfills once:
    import 'reflect-metadata'

    # Typescript users

    Please enable decorators support in your tsconfig.json, as well as es2019.arrays

    {
  "compilerOptions": {
    "emitDecoratorMetadata": true,
    "experimentalDecorators": true,
    "lib": [
          "es2019.array"
        ]
    }
}

    This library has been compiled with Typescript 3.7+. As a result of this issue (opens new window), projects using typescript < 3.7 won't compile unless the following is specified in your tsconfig.json:

    "compilerOptions": {
    "skipLibCheck": true
}
